### Source Academy

**RA1 & RA2** were fully MCQ.  
Especially **RA2** this year was *super tricky*, with a median of **43/95**.

**Midterm** was MCQ + code implementation (on paper).  
Fairly hard ‚Äî you needed deep understanding to move fast. I lost:
- 9 points on the **T-diagram** (I didn‚Äôt get it yet)
- 3 more points on other MCQs

**Practical Assessment** was fully code implementation on Source Academy:
- 7 problems
- 100 minutes
- public tests for debugging + private tests for grading  
Last 3 were hard, but I finished everything in **70 minutes**.

**Final** was MCQ + code implementation too.  
This year‚Äôs final felt easier than previous years.

---

### Conclusion
If you‚Äôre completely new to programming, you need to learn this course **VERY carefully**.  
The course content is excellent for teaching how to think and design code properly.
